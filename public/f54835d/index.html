全连通内网守护程序

西楼
2021-06-10

项目背景

一个组织的敏感接口不能暴露在外网。

而若以网关的形式转发流量，那么瓶颈会出现在网关上。

因此需要开发出支持任意两台机器都可以通信的软件，并且该软件可以支持 mesh 连接。

方案描述

  • [ ] 支持配置文件进行配置
  • [ ] 将 WireGuard 拓展支持 Mac 地址
  • [ ] mesh 连接
  • [ ] 使得机器支持热部署
  • [ ] 让任意两台 NAT 后的机器直接相连

时间规划

  • 熟悉技术栈
    • 7.1 - 7.25
    • 学习 go 语言，阅读 wireguard 的 go 语言实现，了解 mesh 连接如何实现等等
  • 重构 wireguard 的 go 语言实现
    • 7.25 - 8.20
    • 将 wireguard 中所需功能提取出来，并修改使其支持 mesh
  • 进行性能测试
    • 8.20 - 8.25
    • 做性能测试，并修改源代码
  • 书写文档
    • 8.25 - 8.30
    • 书写相关的文档
  • 增加 Dashboard
    • 若有空余时间
    • 让机器可以可视化监控状态
